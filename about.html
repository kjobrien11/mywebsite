About this project.


This project was designed as an exploratory endeavor into JavaScript. Having minor prior experience in JS, my aim was to create something useful while also pushing myself to think creatively and learn and a new language.


The idea for the “terminal” based interface came one night after hours of git commits in the terminal. After creating the basic structure of the website, I was lost as to which direction to pursue to stylize it. Then it hit me: the terminal. I’ve always loved design simplicity and thought that pursing a terminal-based UI would straddle the line between design simplicity and creativity.
